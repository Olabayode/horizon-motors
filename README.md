## Overview
This repository contains the frontend code for a multi-page automotive-themed website built using HTML, CSS, and JavaScript.
The code focuses on clean structure, reusable components, responsive layout design, and simple UI interactions.

This project includes:
- A Home page (index.html)
- A services details page (detail.html)
- A stylesheet fit for both pages
- Also a scripts file for handling all interactive features
